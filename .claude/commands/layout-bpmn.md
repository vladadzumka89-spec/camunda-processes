📐 Перевірити та виправити візуальну розкладку BPMN-діаграми

## Workflow

### Крок 1: Визначити файл
Якщо $ARGUMENTS містить шлях до .bpmn файлу — використати його.
Інакше запитати через AskUserQuestion: "Який BPMN файл перевірити?" з варіантами з `bpmn/` директорії.

### Крок 2: Перевірка
Запустити:
```bash
python3 bpmn_layout_checker.py "$FILE"
```

### Крок 3: Звіт
Показати результати перевірки у форматі:

**Результати перевірки `<filename>`:**
- Перекриття елементів: N
- Перекриття стрілок: N
- Boundary events не на межі: N
- Елементи поза lane: N
- Стрілки через елементи: N
- **Всього: N проблем**

### Крок 4: Рішення
Якщо 0 проблем — повідомити "Розкладка коректна, виправлення не потрібні" і завершити.
Інакше запитати через AskUserQuestion: "Знайдено N проблем. Виправити автоматично?"
Варіанти:
- "Так, виправити (overwrite)" — перезаписати оригінал
- "Так, виправити (в новий файл)" — зберегти поруч з суфіксом `-layout`
- "Ні, тільки звіт" — завершити

### Крок 5: Авто-розкладка
Запустити відповідну команду:
```bash
# Overwrite:
python3 bpmn_auto_layout.py "$FILE" --verbose

# Новий файл:
python3 bpmn_auto_layout.py "$FILE" -o "${FILE%.bpmn}-layout.bpmn" --verbose
```

### Крок 6: Повторна перевірка
Запустити:
```bash
python3 bpmn_layout_checker.py "$OUTPUT_FILE"
```

### Крок 7: Підсумок
Показати:

**Підсумок:**
- До: N проблем
- Після: M проблем
- Виправлено: N-M проблем
- Файл: `<output_path>`
